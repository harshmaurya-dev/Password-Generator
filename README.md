# 🔐 Password Generator

A simple and responsive **Password Generator** built using **HTML, CSS, and JavaScript**.

## 🚀 Features

- Generate strong random passwords
- Adjustable password length
- Include numbers option
- Include symbols option
- One click copy password feature
- Clean and responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📸 Preview

Password Generator web application with a modern dark interface.

## 📂 How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. Generate your secure password
